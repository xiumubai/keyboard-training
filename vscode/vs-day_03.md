## 全局搜索

shift + command + f

commnd + up/down 切换到下方选择区域

通过 jk 上下选择文件

l 回到选择的文件

toggel query detaisls: shift + command + j

## 工作区搜索

command + t
搜索函数 搜索变量

## 文件内搜索

shift + command + o

@ + : 对函数和变量进行排序(函数在上，变量在下面)

## 命令搜索

command + shift + p

## 文件搜索

command + p 可以搜索切换文件

输入>：切换成搜索命令
输入#：切换成工作区搜索函数和变量
输入@：切换成文件内搜索函数和变量

ctrl+tab 切换打开的文件 tab
